# Syndicate Motors - Premium Car Dealership Website

A modern, mobile-first car dealership website for Syndicate Motors in Hyderabad, India. Built with React, TypeScript, and Tailwind CSS. Features smooth animations, clean design, and optimized user experience with popular Indian car models.

```javascript

  extend: {
    colors: {
      primary: {
        // Your custom colors
      }
    }
  }
```

### Modify Content
Update the data arrays in each component file to change:
- Car listings
- Services
- Testimonials
- Contact information
